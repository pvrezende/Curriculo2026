# Currículo 2026 — Paulo Rezende

Projeto atualizado para apresentar um currículo profissional com foco em:

- melhor leitura por recrutadores
- conteúdo otimizado para ATS
- exportação simples para PDF
- visual moderno e limpo
- versão adequada para vagas presenciais e remotas

## Arquivos

- `index.html` → estrutura principal do currículo
- `style.css` → estilos visuais, responsividade e impressão
- `reset.css` → reset básico do projeto
- `main.js` → alternância de tema e atalho para impressão/PDF

## Como testar

1. Extraia o `.zip`
2. Abra a pasta no VS Code
3. Abra o arquivo `index.html` no navegador
4. Clique em **Baixar / Imprimir PDF** para salvar em PDF

## Melhorias aplicadas

- conteúdo reescrito em linguagem profissional
- inclusão de palavras-chave importantes para ATS
- remoção do excesso de elementos visuais que atrapalham recrutadores
- layout mais corporativo
- organização melhor de experiência, projetos e habilidades

## Próximos passos recomendados

- adaptar uma versão específica para cada vaga
- manter LinkedIn com o mesmo texto-base do currículo
- destacar no portfólio projetos com Node.js, AWS e banco de dados
